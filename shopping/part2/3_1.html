发送邮件

  • Django中内置了邮件发送功能，被定义在django.core.mail模块中
  • 发送邮件需要使用SMTP服务器，常用的免费服务器有：163、126、QQ，下面以163邮件为例
  • 注册163邮箱itcast88，登录后设置

  • 在新页面中点击“客户端授权密码”，勾选“开启”，弹出新窗口填写手机验证码

  • 填写授权码

  • 提示开启成功

  • 打开settings.py文件，点击下图配置
EMAIL_BACKEND = 'django.core.mail.backends.smtp.EmailBackend'
EMAIL_HOST = 'smtp.163.com'
EMAIL_PORT = 25
#发送邮件的邮箱
EMAIL_HOST_USER = 'itcast88@163.com'
#在邮箱中设置的客户端授权密码
EMAIL_HOST_PASSWORD = 'python808'
#收件人看到的发件人
EMAIL_FROM = 'python<itcast88@163.com>'
  • 在tt_user/views.py文件中新建视图send
from django.conf import settings
from django.core.mail import send_mail
from django.http import HttpResponse
...
def send(request):
    msg='<a href="http://127.0.0.1/active/" target="_blank">点击激活</a>'
    send_mail('注册激活','',settings.EMAIL_FROM,
              ['itcast88@163.com'],
              html_message=msg)
    return HttpResponse('ok')
  • 在tt_user/views.py文件中新建视图active
def active(request):
    return HttpResponse('激活')
  • 在tt_user/urls.py文件中配置
    url(r'^send/$',views.send),
    url(r'^active/$',views.active),
  • 启动服务器，在浏览器中输入如下网址
http://127.0.0.1:8000/send/
