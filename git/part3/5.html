开发管理

  • 上面的操作，只有我们在加入项目的第一天需要进行，只操作一次就够了
  • 接下来的操作，是我们每天开发中都要进行的操作，这是必须做到熟练操作的命令
  • 当前用户以zhujiao分支进行开发
git checkout zhujiao
  • 本地仓库分为三部分：工作区，暂存区，仓库区，其中暂存区、仓库区是版本库部分
  • 使用IDE如sublime、pycharm等打开项目目录
