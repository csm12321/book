2. pycharm的vim插件安装

方法(一)：在线安装

File -> Settings -> Plugins -> Install JetBrains Plugins

输入 ideavim -> Install Plugin

重新启动pycharm

方法(二)：手动导入

插件地址:打开链接

如果以上网址打不开，可以打开备用下载网址:打开链接

File -> Settings -> Plugins -> Install plugin from disk -> ideavim-0.41-290.zip -> OK

重新启动pycharm
